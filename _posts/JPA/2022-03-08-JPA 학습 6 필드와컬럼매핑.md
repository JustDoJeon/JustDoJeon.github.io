# 필드와 컬럼 매핑


#### 매핑 정리

@Column : 컬럼 매핑
@Temporal : 날짜 타입 매핑
@Enumerated : enum 타입 매핑
@Lob : BLOB, CLOB 매핑 (varchar 보다 훨씬 큰 크기가 필요 할 때 사용 ) 
@Transient : 메모리에서만 쓰고 디비에는 상관없이 사용할때 사용
